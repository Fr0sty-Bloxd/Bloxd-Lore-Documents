# Bloxd Lore Documents

# 1.Creation
Bloxd was not orginally created by Arthur, it was actually created by a user known as Weiqing. Bloxd.io was later handed to Arthur (Which caused more gamemodes to be added).
Early Bloxd was just a parkour game, it was fully named Bloxdhop.io before being turned into  a sandbox game.
On the 23rd of October 2021, the second gamemode was added, Cube warfare. This was the first gamemode to ever include pvp and added guns (gamemode was later replace by clutch).
