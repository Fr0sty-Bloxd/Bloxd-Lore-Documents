# Bloxd Lore Documents

DISCLAIMER: Not all updates are included only the most important ones are, the actual lore its towards the end.

# 1.Creation
Bloxd was not orginally created by Arthur, it was actually created by a user known as Weiqing. Bloxd.io was later handed to Arthur (Which caused more gamemodes to be added).
Early Bloxd was just a parkour game, it was fully named Bloxdhop.io before being turned into  a sandbox game.
On the 23rd of October 2021, the second gamemode was added, Cube warfare. This was the first gamemode to ever include pvp and added guns (gamemode was later replace by clutch).

# 2.2022
On January 7th of 2022 the famous *"Bob"* type skin was added to bloxd.
Worlds, Inventories, Protectors, Survival, Peaceful, Etc. were added to the game.
More famous gamemodes were added to Bloxd such as Oneblock, Bedwars, and greenville
Along with multiple commands, blocks, and bug fixes added into the game.
As a troll an Iron Watermelon entity was added to the game and would "Jumpscare" players if they go outside 6,000 blocks from spawn (due to chunk generation being non perfected), Though these are rumors.

# 3.2023
New biomes were added to the game to increase Bloxd's popularity with new ice rivers being added.
Murder mystery was added to the game as a limited time gamemode.
Press to get boards were added to the game with explosives, seeing this many players abused this glitced to get an infinite amount of any material they needed.
Skywars was added and prison was a limited time gamemode that was sadly removed due to a vote between plots or prison.

# 4.2024
Pirates was readded in a gamemode vote against survival royale.
